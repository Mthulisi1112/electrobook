# ElectroBook - Find Trusted Electricians

![ElectroBook Logo](public/images/logo.png)

ElectroBook is a professional marketplace web application that connects customers with licensed, verified electricians. Users can browse services, compare electricians, read reviews, and book appointments instantly.

## 🚀 Features

### For Customers

- **Search & Browse**: Find electricians by service type, location, or name
- **Service Filtering**: Filter electricians by service categories (Emergency Repair, Wiring, Lighting, etc.)
- **Booking System**: Schedule appointments with real-time availability
- **Reviews & Ratings**: Read customer reviews and rate electricians after service
- **User Profiles**: Manage personal information and booking history

### For Electricians

- **Business Profile**: Create and manage professional profile with business details
- **Service Management**: List services offered with custom pricing
- **Availability Calendar**: Set and manage available time slots
- **Booking Management**: View, confirm, and manage customer bookings
- **Reviews Dashboard**: Track customer ratings and feedback

### For Administrators

- **User Management**: Manage all users, verify electricians, handle roles
- **Service Management**: Create and manage service categories
- **Reports**: Generate analytics and reports on platform activity
- **Verification System**: Verify electrician credentials and licenses

## 🛠️ Tech Stack

- **Framework**: Laravel 12.x
- **Frontend**: Blade Templates, Tailwind CSS, Alpine.js
- **Database**: SQLite (development) / MySQL (production)
- **Authentication**: Laravel Breeze / Jetstream
- **Image Storage**: Laravel Storage (local/production)
- **UI Components**: Custom Tailwind components
- **Icons**: Font Awesome, Heroicons

## 📋 Prerequisites

- PHP >= 8.1
- Composer
- Node.js & NPM
- SQLite or MySQL
